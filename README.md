# small-yan-utils

# 安装命令

```s
# npm 安装
npm i
# yanr 安装
yarn
```

# 打包项目

```s
# npm
npm run build
# yarn
yarn build
```

# jest 测试命令

```s
# npm
npm run test
# yarn
yarn test
```
